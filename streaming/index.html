A/V Streaming Project Introduction

Systemwide Digital Library Services will be working with @mire to provide configuration support for setting up AWS S3 and AWS CloudFront to support a Cloud based streaming solution to replace Kaltura. @mire will provide a setup for customization to CSU DSpace to support transcoding, storage and the streaming of video content from Amazon CloudFront. All existing media content both in the existing Kaltura server and only in DSpace will be migrated to the new cloud based solution upon completion. If you have multi-media projects that are waiting for a streaming solution, you are welcome to being uploading content into DSpace today and as this solution becomes available your content will be transcoded automatically.
